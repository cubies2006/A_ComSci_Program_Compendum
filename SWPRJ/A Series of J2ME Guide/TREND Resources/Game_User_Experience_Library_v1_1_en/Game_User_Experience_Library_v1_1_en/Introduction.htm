Game User Experience Library

 

Version 1.1; April 28, 2006

Introduction

Developing good games is not an easy task. Good playability and usability is often a result of hard work in all design phases. This library contains game experience-related articles, information, and tools to assist game developers in developing usable games.

    • Top ten guidelines contains one-page summarizations of the most important game usability guidelines.

    • Checklists are a quick way of getting started on evaluating the usability of mobile games.

    • Main menu design contains visual presentations of main menu design recommendations.

    •  Usability guidelines contains detailed design recommendations about different topics in game development.

    •  Articles contains various game experience-related articles that have been published at Forum Nokia.

    • Links and references contains links and references to other information sources.
