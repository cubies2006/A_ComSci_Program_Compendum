Bluetooth connection menu

    • Do not use technical terms like “Create server” and “Connect to server.” Use “Create game” and “Join game” instead.

    • Users do not need to know the Bluetooth name of a server device. The name of the game should be queried when a user selects “Create game” from the Bluetooth menu. Reasonable default settings should be provided for the users, so that they only need to accept possible dialogs or queries.

    • Provide instructions for establishing the Bluetooth game. These should be available in the Bluetooth connection menu.
